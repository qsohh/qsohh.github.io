## Local server

调试的时候浏览器一般来讲会禁止直接读取本地`json`文件，这情况下就需要使用本地服务器。一个比较简单的办法是通过`Python 3`：

```bash
python -m http.server
```

之后在浏览器中访问地址 [http://localhost:8000](http://localhost:8000) 即可。
